Class Hierarchy

  • object: The most base type
    • classEditor.AbstractBar: Classe abstraite pour les barres
      • classEditor.Arc: Classe pour une barre de type arc
      • classEditor.Barre: Classe pour une barre de type segment
      • classEditor.MBarre: Classe pour une barre de type barre à noeud multiple
      • classEditor.Parabola: Classe pour une barre de type parabole
    • classEditor.AbstractNode: Classe abstraite pour les noeuds
      • classEditor.ArcNode: Classe pour un noeud d'arc
      • classEditor.Node: Classe contenant pour un noeud simple
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • classEditor.MyError: Surcharge de la classe d'erreur
    • classEditor.CMenuChar1: Classe pour les menus contextuels du chargement sur un arc
    • classEditor.CMenuChar2: Classe pour les menus contextuels du chargement sur une barre
    • classEditor.CMenuEmpty: Classe pour les menus contextuels en dehors d'un item
    • classEditor.CMenuNode1: Classe pour les menus contextuels d'un noeud d'arc
    • classEditor.CMenuNode2: Classe pour les menus contextuels d'un noeud simple
    • classEditor.Case: Classe pour un cas de charge
    • classEditor.Char: Classe de base pour un chargement
      • classEditor.CharArc: Classe pour un chargement sur un arc
      • classEditor.CharBar: Classe abstraite pour les chargements appliqués sur barre unique
        • classEditor.CharFp: Classe pour un chargement nodal
        • classEditor.CharTh: Classe pour un chargement thermique
        • classEditor.CharTr: Classe pour un chargement triangulaire
      • classEditor.CharNo: Classe pour un chargement nodal
      • classEditor.CharPp: Classe pour un chargement de type poids propre
      • classEditor.CharQu: Classe pour un chargement uniforme
    • classEditor.Combi: Classe pour une combinaison de cas de charge
    • classEditor.DataEditor: Classe pour les données rdm d'une étude
    • classEditor.Editor: Classe unique pour l'éditeur des données
    • classEditor.EmptyRdm: Classe Rdm pour une nouvelle étude
    • classEditor.Material: Classe pour un matériau
    • classEditor.Section: Classe pour une section droite
